# Techviora Company Website - Product Requirements Document

## Original Problem Statement
Build a professional website for Techviora company showcasing their product development services with specific content sections including hero, services, pricing, case studies, process, and contact information.

## User Personas
- **Primary**: Startup founders and business owners looking for complete product development services
- **Secondary**: Growing businesses needing MVP development and scaling solutions

## Core Requirements
- Modern, conversion-optimized landing page
- Lime-green brand color with dark backgrounds
- Smooth animations and micro-interactions
- WhatsApp integration for consultation booking
- Contact form with email integration
- Fully responsive design
- Professional agency-quality appearance

## Architecture & Tech Stack
- **Frontend**: React 19, TailwindCSS, Shadcn/UI components
- **Styling**: Custom animations, glass-morphism effects
- **Components**: Modular architecture with reusable components
- **Backend**: Static site (no backend required initially)
